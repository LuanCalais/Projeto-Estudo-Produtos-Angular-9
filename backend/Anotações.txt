mkdir nomeDaPasta - Cria uma nova pasta 
cd nomeDaPasta - navega pelas pastas
npm init -y - Cria um arquivo package.json(para colocar dependências)
npm i json-server - Cria um json-server (lê um arquivo que possui um json e cria uma API baseada nesse json)
npm start - dentro da pasta backend inicia a API

ATALHOS
alt + shift + prabaixo - Duplica campos selecionados
alt + prabaixa - movimenta itens selecionados
